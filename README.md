# Yuugubot
Now the bot is adapted only for Yandex Alice

## Installation
Dependencies:
- systemd
- git *
- docker
- python3

Run these commands:
```
$ git clone https://github.com/RotiVP/yuugubot
$ cd yuugubot/vps
# install.sh
```

## Launching
In yuugubot/vps:
```
# start.sh
```

## Uninstallation
In yuugubot/vps:
```
# uninstall.sh
```

## Updating
Do the following steps:
- uninstall the bot
- install the bot

## Note
The bot listens on 443 port
